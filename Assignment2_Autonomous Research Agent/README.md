# 🤖 Autonomous Research Agent (LangChain + Groq)

An AI-powered research agent that automatically gathers information from the web and Wikipedia, then generates a structured academic report in PDF format.

---

## 🚀 Features

- 🔍 Web search using Tavily API
- 📚 Wikipedia knowledge integration
- 🧠 Autonomous research using LangChain agents
- 📝 Structured academic report generation
- 📄 PDF export with proper formatting
- 📂 Automatic file saving with timestamp

---

## 🛠️ Tech Stack

- Python
- LangChain
- Groq (LLM)
- Tavily API
- Wikipedia API
- ReportLab (PDF generation)
